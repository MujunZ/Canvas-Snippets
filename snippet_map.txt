Below is a list of all the methods and properties of the Canvas' Context followed by its respective file name and the folder that contains them.

How to Use this Map:

	FOLDER NAME
		method/property name - file_name

	'p_' in the file_name means it's a Context Property
	'm_' in the file_name means it's a Context Method

Note:
	1. The Canvas' Context is set to "ctx" and should not be changed
	2. There are some MISC snippets, take a look at the MISC section
